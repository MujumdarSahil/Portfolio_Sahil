# Sahil Mujumdar — Netflix Portfolio

A Netflix-themed personal portfolio for **Sahil Mujumdar**, Data Scientist. All content is stored locally in `src/data/portfolioData.ts` — no external CMS required.

## Features

- Netflix-style intro and profile selection
- Work experience, projects, skills, certifications
- Reading shelf (AI & history), music, contact
- Resume download from `/Sahil_Mujumdar_Resume.pdf`

## Run locally

```bash
npm install
npm start
```

## Deploy

```bash
npm run build
```

Deploy the `build/` folder to any static host (Vercel, Netlify, AWS S3, Cloudflare Pages, etc.).

## Update content

Edit `src/data/portfolioData.ts` for experience, projects, skills, certifications, and contact details.
